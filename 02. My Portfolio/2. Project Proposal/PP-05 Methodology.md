The project will utilise the agile development approach to support adaptability and progress in increments. Work will be done in short development cycles, allowing for regular review, testing, and refinement based on evaluation.

### Project Phases

- **Investigation:** Identify user needs, define project requirements, and research existing study planning tools.
    
- **Design:** Develop interface, database structures, and user interaction flows.
    
- **Implementation:** Code and integrate core application features, including user management and study scheduling.
    
- **Testing:** Perform functional testing, usability checks, and resolve identified issues.
    
- **Finalisation:** Prepare the app for submission and complete all supporting documentation.

	### Project Deliverables

Upon completion of the Anagram project, the following deliverables will be produced:

- **Working Application:**  
	A fully functional progressive web application that meets WC3 PWA standards, supports offline use, and operates across desktop and mobile browsers.
    
- **Source Code Repository:**  
    A GitHub repository containing the complete application source code, demonstrating clear structure and regular, descriptive commits.
    
- **Project Portfolio:**  
    A digital portfolio documenting the planning, development, and evaluation of the project, including evidence of consistent improvement.
    
- **Technical Documentation:**  
    Concise documentation outlining system design, security considerations, and implementation details.
    
- **User Guide:**  
    A clear guide explaining how to use the application.
    
- **Evaluation reports:**  
    Documentation summarising testing outcomes, user feedback, and reflection against success criteria.
